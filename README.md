# ELUX — Modern Shopping Experience

ELUX is a premium, high-performance e-commerce frontend built using **Next.js 14 (App Router)**, React, Tailwind CSS, and TypeScript. It features a fully responsive, modern user interface, smooth route transitions, client-side state management, and an optimized, SEO-friendly architecture.

---

## 🚀 Live Demo
You can view the live deployment of this project here:  
👉 **[elux-fixed.vercel.app](https://elux-fixed.vercel.app/)**

---

## ✨ Features

*   **Premium Modern UI/UX:** Styled using Tailwind CSS with elegant layouts, interactive hover effects, and clean grid components.
*   **Next.js App Router & Server Prerendering:** Fully optimized using nested route groups and React `Suspense` boundaries for high-performance builds.
*   **Flexible Product Catalog:** Sort and filter through high-quality visual items under multiple computing, photography, audio, and wearables categories.
*   **Responsive Images:** High-resolution product showcase powered by fast, optimized Pexels CDN image integration.
*   **Interactive Admin Dashboard:** Dedicated `/admin` route for verifying and viewing active data parameters.
*   **TypeScript Verified:** Fully type-safe codebase using custom product schemas and strict React structure guidelines.

---

## 🛠️ Tech Stack

*   **Framework:** Next.js (App Router)
*   **Styling:** Tailwind CSS
